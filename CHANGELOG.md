# Changelog

All notable changes to the Algan Studio website project.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added

#### Core Architecture
- Astro 4.x with static site generation
- React 18 for interactive components
- TypeScript for type safety
- Tailwind CSS 3.x for styling
- Framer Motion for animations
- GSAP for scroll animations

#### Design System
- CSS custom properties (design tokens)
- Typography scale (Inter + Space Grotesk)
- Color palette (brand + semantic)
- Spacing system
- Shadow system
- Animation utilities
- Dark mode support ready

#### UI Components (29)
- Avatar, Badge, Breadcrumb, Button
- Card, Checkbox, Divider, Input
- Marquee, ProcessStep, ProgressBar, Radio
- SectionHeader, Select, ServiceCard, Skeleton
- TestimonialCard, Textarea, Accordion, Carousel
- Lightbox, Modal, StatsCounter, Switch
- Tabs, Toast, Tooltip

#### Global Components (6)
- Header with mega menu navigation
- Footer with newsletter signup
- Mobile menu with slide-out drawer
- SEO component with meta tags
- WhatsApp floating button
- Newsletter signup form

#### Pages (52+)
- Homepage with 8 sections
- 9 service pages with pricing and FAQ
- 10 case study pages with galleries
- 20 blog posts with TOC
- About page with team grid and timeline
- Contact page with validated form
- Pricing page with comparison tables
- Privacy policy page
- Terms of service page
- Custom 404 error page

#### Data Layer
- Site configuration constants
- Services data (9 services)
- Case studies data (10 studies)
- Blog posts data (20 posts)
- Team members data (12 members)
- Pricing plans data (9 plans)

#### SEO & Performance
- Automatic sitemap generation
- robots.txt configuration
- Schema.org structured data
- Open Graph meta tags
- Twitter Card support
- Image optimization with Sharp
- Code splitting
- Prefetching
- Lazy loading

#### Accessibility
- WCAG 2.1 AA compliance
- Semantic HTML
- ARIA labels
- Focus management
- Keyboard navigation
- Reduced motion support
- Color contrast compliance

#### Developer Experience
- ESLint configuration
- Prettier configuration
- TypeScript strict mode
- Component documentation
- Comprehensive README
- Deployment guides

### Configuration Files
- astro.config.mjs
- tailwind.config.mjs
- tsconfig.json
- package.json
- .env.example
- vercel.json
- netlify.toml
- eslint.config.js
- .prettierrc
- .gitignore
